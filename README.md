# tic-tac-toe

## Introduction
A sleek version of the classic game tic-tac-toe.


## Technologies

- Javascript
- HTML
- CSS

## Features

- Fully playable tic-tac-toe game. 
- Show current turn beneath board.
- On win, freeze board, show winner and update win count.
